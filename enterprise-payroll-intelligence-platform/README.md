# Enterprise Payroll Intelligence Platform

## Overview
This project simulates an enterprise-grade payroll analytics system inspired by ADP-scale architecture.

## Key Features
- Batch + Real-time data processing
- Medallion Architecture (Bronze, Silver, Gold)
- ML-based attrition prediction
- Payroll anomaly detection
- Power BI dashboards
- Data governance & security

## Tech Stack
- PySpark
- SQL
- Microsoft Fabric / Azure
- Delta Lake
- Power BI

## Architecture
Refer to /1_architecture for detailed diagrams.

## Business Problem
Organizations need real-time insights into payroll, employee costs, and attrition risks.

## Solution
A scalable Lakehouse platform that processes HR data and provides actionable insights.
